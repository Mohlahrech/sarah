Website Sale Address City Selection

Create Cities and select city for billing or shipping address

Few Words About The Website Sale Address City Selection:

This Module provide you a option to select your city for billing or shipping address in website. These cities are created in back-end.

Work Flow: How the "Website Sale Address City Selection" works
Create State and City:
  • Open the countries and create states and cities
Create Cities in a state
Select a City
  • As per the selected country and state select your city for billing or shipping address
Enter a City Manually
  • When a list of cities is not available for selected state, you can manually enter your city
